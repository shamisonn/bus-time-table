# Bus Time table

これはTDUにおける鳩山キャンパスのバス時刻表アプリです。

go https://shamisonn.github.io/bus-time-table/

# Feature

- PWA対応済み。
- オフライン対応済み。
- デザインはもうだめ。
- Yahoo乗り換えアプリを参考にしました。

```
$ yarn run start
```

- オフライン対応としてworkboxを使った。
- ついでにpwa化した。
